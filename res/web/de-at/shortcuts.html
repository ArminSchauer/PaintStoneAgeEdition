Sprache: Deutsch / English

Tastenkombinationen

1.) Datei

STRG + N Neue Datei
STRG + O Datei öffnen/laden
STRG + S Datei speichern
STRG + UMSCHALT + S Datei speichern unter

2.) Bearbeiten

STRG + X Selektierte Elemente löschen und in die Zwischenablage kopieren
STRG + C Selektierte Elemente in die Zwischenablage kopieren
STRG + V Elemente aus der Zwischenablage einfügen
ENTF Selektierte Elemente löschen

3.) Selektierung

STRG + A Alle Elemente selektieren
STRG + UMSCHALT + A Keine Elemente selektieren

4.) Ansicht

STRG + PLUS Hineinzoom
STRG + MINUS Herauszoomen
STRG + NUMPAD-0 Zoom zurücksetzen auf 100%
