Language: Deutsch / English

Keyboard shortcuts

1.) File

CTRL + N New file
CTRL + O Open/load file
CTRL + S Save file
CTRL + SHIFT + S Save file as

2.) Edit

CTRL + X Delete selected elements and copy them to clipboard
CTRL + C Copy selected elements to clipboard
CTRL + V Paste elements from clipboard
DEL Delete selected elements

3.) Select

CTRL + A Select all
CTRL + SHIFT + A Select none

4.) View

CTRL + PLUS Zoom in
CTRL + MINUS Zoom out
CTRL + NUMPAD-0 Zoom reset to 100%
